# Настройка списка администраторов

## Инструкция

Создайте файл `.env.local` в корне проекта и добавьте следующую переменную окружения:

```env
ADMIN_ALLOWLIST=Дибров Дмитрий Алексеевич;Забудько Алексей Викторович;Рыжих Владислав Васильевич;Быков Сергей Дмитриевич;Бобович Павел Александрович
```

## Команда для создания файла (PowerShell)

**ВАЖНО:** Из-за проблем с кодировкой в PowerShell, создайте файл вручную:

1. Откройте редактор (например, VS Code или Notepad++)
2. Создайте новый файл с именем `.env.local` в корне проекта
3. Скопируйте и вставьте следующую строку:

```
ADMIN_ALLOWLIST=Дибров Дмитрий Алексеевич;Забудько Алексей Викторович;Рыжих Владислав Васильевич;Быков Сергей Дмитриевич;Бобович Павел Александрович
```

4. Сохраните файл с кодировкой UTF-8 (без BOM)

## Важно

- После создания `.env.local` перезапустите dev-сервер (`npm run dev`)
- ENV переменные загружаются только при старте приложения
- Разделитель между ФИО — точка с запятой (`;`)
- Авторизация нечувствительна к регистру

## Добавление новых администраторов

Чтобы добавить новых админов, добавьте их ФИО в переменную `ADMIN_ALLOWLIST` через точку с запятой:

```env
ADMIN_ALLOWLIST=Дибров Дмитрий Алексеевич;Забудько Алексей Викторович;Рыжих Владислав Васильевич;Быков Сергей Дмитриевич;Бобович Павел Александрович;Новый Админ Иванович
```
